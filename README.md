# Dobavna veriga zdravil (diplomska)

Prototip za sledljivost zdravil od proizvajalca do lekarne: preverljive poverilnice (Walt.id), metapodatki na IPFS in revizijska sled na verigi blokov.

## Zagon

Pred prvim zagonom kopiraj `src/.env.example` v `src/.env` in po potrebi vnesi ključe (npr. Pinata).

```bash
docker compose up -d
```

Aplikacija: **http://localhost:20000**
